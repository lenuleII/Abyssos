## ABYSSOS

The cryocapsule lid opens, and air rushes into my lungs, scraping my throat. I am blind, the whole world is an incomprehensible blur. Over time, my mind begins to clear. Right now, I can't remember exactly who I am.

Damn, where am I?

## Building

1. Clone this repo:
```shell
git clone https://github.com/space-wizards/space-station-14.git
```
2. Go to the project folder and run `RUN_THIS.py` to initialize the submodules and load the engine:
```shell
cd space-station-14
python RUN_THIS.py
```
3. Compile the solution:  

Build the server using `dotnet build`.

[More detailed instructions on building the project.](https://docs.spacestation14.com/en/general-development/setup.html)
